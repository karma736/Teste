# Chat Público Simples

Aplicação de chat público com Node.js, Express e SQLite. As mensagens expiram após 2 horas e o front-end permite lembrar o nome do usuário via `localStorage`.

## Como executar
1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor:
   ```bash
   npm start
   ```
3. Acesse no navegador:
   [http://localhost:3000](http://localhost:3000)
