Publicação simples

Hospede no GitHub Pages

Nenhuma configuração de backend necessária.

  1. Habilite o GitHub Pages na aba Settings → Pages escolhendo a branch main.
  2. Faça commit e push deste código. A página será publicada automaticamente.
  3. Compartilhe a URL pública para que todos possam ler e postar mensagens que expiram em 1 hora.

Tecnologias usadas

  • HTML estático no GitHub Pages
  • Gun.js como banco distribuído sem servidor
  • JavaScript moderno para expiração automática
