Como funciona

Passos para deixar público

Faça commit deste código e habilite o GitHub Pages para publicar automaticamente.

  1. Habilite o GitHub Pages na aba Settings → Pages escolhendo a branch main.
  2. Suba o código para o GitHub; o workflow padrão publica o site em poucos minutos.
  3. Visite a URL pública gerada pelo Pages e comece a registrar entradas.

Tecnologias usadas

  • HTML estático hospedado no GitHub Pages
  • IndexedDB para persistência local sem servidor
  • JavaScript moderno para operações de CRUD
